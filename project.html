Jogo do Mario


Criei um jogo do Mario para praticar HTML, CSS e JS. Busquei um projeto que fosse simples e encontrei no canal Manual do Dev. Foi muito satisfatório conseguir chegar ao final. Tinha momentos em que não conseguia encontrar os erros, até uma pequena vírgula faz toda diferença. O jogo foi criado com uma estrutura básica de HTML, utilizando o CSS para fazer o estilo do jogo e suas animações. Para as animações que precisavam cumprir uma função específica, utilizamos o JavaScript. Nele, programei o movimento do cano do Mario e os próprios movimentos.



Game Pokémon


Neste projeto de jogo, o desenvolvedor utilizou HTML, CSS e JavaScript para criar uma experiência interativa onde o jogador controla o Ash em busca de Pokémon, como Pikachu, Zubat e Charmander. O tempo é crucial e se todos os Pokémon não forem encontrados a tempo, o jogo termina. Além disso, uma trilha sonora customizável e controles de música interativos foram implementados para proporcionar uma experiência ainda mais envolvente.




Calculadora IMC


Esta calculadora interativa foi desenvolvida utilizando HTML, CSS e JavaScript. Ela permite aos usuários calcular o Índice de Massa Corporal, uma medida importante para avaliar a saúde física. Com uma interface intuitiva e fácil de usar, os usuários podem inserir seu peso e altura para obter instantaneamente o IMC e uma indicação do estado de sua saúde. Esta aplicação é uma ferramenta valiosa para promover a conscientização sobre a importância do peso e da saúde corporal.
